# Reforge Linux v0.3-dev

**Status:** Development preview, not production release

**Tagline:** Old hardware. New purpose.

## Overview

Reforge Linux is an old-PC repurposing Linux project with guided setup profiles. The v0.3-dev milestone focuses on completing the first practical profile set, improving documentation, and preparing the project for future ISO work.

This is a development preview intended for testing, documentation, and portfolio progress. It is not a production-ready server OS release.

## Highlights

- Setup Center implemented
- Three profiles implemented
- VM testing completed
- Documentation improved
- Branding plan added
- ISO build plan added

## Working Profiles

- File Server using Samba
- Ad-Blocking DNS Server using Pi-hole
- Media Server using Jellyfin

## Testing Summary

Testing was completed in an MX Linux 25.2 XFCE x64 VirtualBox VM.

Verified behavior:

- File Server tested from Windows as a mapped `Z:` drive
- Jellyfin tested from a Windows browser
- Pi-hole DNS blocking tested using `nslookup`
- `doubleclick.net` returned `0.0.0.0 / ::` through Pi-hole DNS

## Known Limitations

- No custom ISO released yet
- Real old-hardware testing pending
- Pi-hole needs a stable IP or DHCP reservation for real use
- Not production-ready
