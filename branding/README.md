# Reforge Linux Branding

## Brand Overview

Reforge Linux is built around the idea of old hardware being reforged into a useful machine. The name reflects a second life for unused computers: instead of sitting idle, an old PC can become a file server, DNS ad blocker, media server, or another practical home utility.

The brand should feel practical, lightweight, and open-source. It should communicate reliability and usefulness without feeling too polished or corporate. Reforge Linux is a hands-on Linux project, so the visual identity should fit naturally with terminal tools, system utilities, and old-PC revival.

## Taglines

- Old hardware. New purpose.
- Give your old PC a job.
- Turn unused computers into useful machines.

## Visual Direction

- Dark theme
- Linux/open-source style
- Old hardware revival
- Forge, spark, and rebuild concept
- Clean terminal/system utility aesthetic
- Not too gamer-looking
- Not too corporate

## Logo Ideas

- An anvil combined with circuit board traces
- An old monitor with a small forge spark
- A letter `R` shaped like a terminal prompt
- A forge flame inside a computer chip
- A minimal Linux-style badge for use in docs, wallpapers, and release pages

## Wallpaper Ideas

- Dark background with subtle circuit lines
- Old PC silhouette in the background
- Glowing forge or spark accent near the center or lower corner
- `Reforge Linux` text as the main focus
- Tagline placement below or near the project name: `Old hardware. New purpose.`
